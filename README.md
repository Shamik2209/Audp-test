# AudioProcessor

## Setup Instructions:

 - Install Python3
 - Install [Django](https://docs.djangoproject.com/en/3.0/intro/install/)
- Install pip requirements : `pip install -r requirements.txt`
- Local Django server: `Django manage.py runserver`
